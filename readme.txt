== Description ==

Dawn is a professional, responsive news theme designed specifically for modern content creators, journalists, and news organizations. Built with WordPress's latest block editor technology, it offers a clean, accessible design that prioritizes readability and user experience.

= Key Features =

* **Full Site Editing (FSE)** - Complete control over your site design using WordPress blocks
* **Responsive Design** - Optimized for all devices (desktop, tablet, mobile)
* **Advanced Block Support** - Custom blocks for enhanced content creation
* **SEO Optimized** - Built with search engine optimization best practices
* **Accessibility Ready** - WCAG 2.1 AA compliant for inclusive web experiences
* **Custom Color Schemes** - Multiple color variations included
* **Typography Options** - Beautiful font combinations for enhanced readability
* **Performance Optimized** - Lightweight and fast-loading
* **Translation Ready** - Fully translatable with included .pot file
* **RTL Language Support** - Perfect for right-to-left languages

= Custom Blocks Included =

* **Load More Block** - Ajax-powered content loading for better user experience
* **Metafield Block** - Display custom fields and metadata
* **Responsive Extensions** - Advanced responsive controls for all blocks

= Layout Options =

* Multiple homepage templates
* Various archive layouts
* Flexible header and footer designs
* Customizable sidebar layouts
* Full-width and boxed layouts

= Browser Support =

* Chrome (latest)
* Firefox (latest)
* Safari (latest)
* Edge (latest)

== Installation ==

= Automatic Installation =

1. Log in to your WordPress admin panel
2. Navigate to Appearance > Themes
3. Click "Add New"
4. Search for "Dawn"
5. Click "Install" and then "Activate"

= Manual Installation =

1. Download the theme zip file
2. Log in to your WordPress admin panel
3. Navigate to Appearance > Themes
4. Click "Add New" > "Upload Theme"
5. Choose the downloaded zip file and click "Install Now"
6. Click "Activate" to enable the theme

= After Installation =

1. Navigate to Appearance > Install Plugins to install the recommended Stepfox Looks plugin
2. Go to Appearance > Site Editor to customize your site
3. Import demo content (if available) via Tools > Import
4. Customize colors, fonts, and layouts via the Site Editor
5. Configure menus and widgets as needed

= Bundled Plugin =

This theme includes the Stepfox Looks companion plugin which provides:
* Advanced block editor functionality
* Custom metafield blocks for displaying custom fields
* AJAX load more features for better pagination
* Comprehensive responsive controls for all blocks
* Enhanced social sharing capabilities
* Block editor extensions and improvements

The plugin is recommended but not required for basic theme functionality.

== Frequently Asked Questions ==

= Is this theme compatible with page builders? =

Dawn is built for WordPress's native block editor (Gutenberg) and Full Site Editing. While it may work with page builders, it's optimized for the block editor experience.

= Can I customize the colors and fonts? =

Yes! The theme includes multiple color schemes and typography options. You can customize everything through the Site Editor under Appearance > Site Editor.

= Is the theme translation ready? =

Absolutely! Dawn is fully translation ready with a complete .pot file included. It supports RTL languages as well.

= Does it work with WooCommerce? =

While the theme is primarily designed for news and blog sites, it should work with WooCommerce with some styling adjustments.

= How do I set up the homepage? =

Go to Appearance > Site Editor, select "Templates" and choose "Front Page" to customize your homepage layout using the available blocks and patterns.

= Can I import demo content? =

Demo content may be available separately. Check with the theme developer for demo content files and import instructions.

= Full Site Editing (FSE) Theme =

As a modern Full Site Editing (FSE) block theme, 'Dawn' utilizes block templates. The standard WordPress functions wp_head(), wp_footer(), and body_class() are handled automatically by the WordPress core during the rendering of these templates and do not need to be called manually.
